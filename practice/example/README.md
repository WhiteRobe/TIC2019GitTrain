### 项目安装指南 Install

**Step 1. 环境依赖**

windows环境下，运行脚本初始化环境.bat，即可自动构建环境依赖(请确保Node.js已在系统下正确配置)

**Step 2. 项目打包**

windows环境下，运行脚本快速打包.bat，可以将前端应用包打包至dist/目录中。
